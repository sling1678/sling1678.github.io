Exercise 2. Spacing Between Slits in a Diffraction Grating from the Direction of First Order Peaks.

A diffraction grating produces a first order peak at \(18^\circ\) for a yellow light of wavelength \(580\, \text{nm}\text{.}\) Find the spacing between the slits.
Hint.
Use condition for the principal peaks.
Answer.
\(1.9\:\mu\text{m}\)
Solution.
Using \(m=1\) in the condition for principal peaks we get
\begin{equation*} a = \dfrac{\lambda}{\sin\theta_1} = \dfrac{0.580\: \mu\text{m}}{\sin\:18^{\circ}} = 1.9\:\mu\text{m}. \end{equation*}
