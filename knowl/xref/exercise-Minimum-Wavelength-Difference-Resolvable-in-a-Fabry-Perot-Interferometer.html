Exercise 4. Minimum Wavelength Difference Resolvable in a Fabry-Perot Interferometer.

You have acquired a Fabry-Perot interferometer with the following specifications: \(d=2.0\text{ cm}\text{,}\) \(R=0.9\text{,}\) \(n=1.1\text{.}\) Find the minimum wavelength difference you can resolve near wavelength of \(\lambda=0.632\ \mu\text{m}\text{.}\)
Hint.
Use formula for resolving power. It helps to calculate finesse first.
Answer.
\(3.16 \times 10^{-7}\ \mu\text{m}\text{.}\)
Solution.
The finesse of the instument is
\begin{equation*} {\cal{F}} = \frac{\pi\sqrt{R}}{1-R} = 29.8. \end{equation*}
This means resolving power will be
\begin{equation*} {\cal{F}}\,\frac{2nd}{\lambda} = 29.8\,\frac{2\times 1.1\times 2.0\times 10^-2}{0.632\times 10^{-6}} = 2.1\times 10^{6}. \end{equation*}
Now, we equate this to \(\lambda/\Delta\lambda\) and solve for \(\Delta\lambda\text{.}\)
\begin{equation*} \Delta\lambda_\text{min} = \frac{ 0.632\ \mu\text{m} }{2.1\times 10^{6}} = 3.16 \times 10^{-7}\ \mu\text{m}. \end{equation*}
