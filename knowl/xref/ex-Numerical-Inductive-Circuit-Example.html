Example 41.16. Numerical Inductive Circuit Example.

A \(50\text{-mH}\) inductor with internal resistance \(0.2\, \Omega\) is connected across the terminals of a \(5\text{-V}\) AC source (i.e. \(V_\text{rms} = \frac{V_0}{\sqrt{2}} = 5\text{ V}\)) of frequency \(60\text{ Hz}\text{.}\) Find
  1. the reactance of the inductor,
  2. the amplitude of impedance of the circuit,
  3. the current amplitude,
  4. the phase constant for current.
Answer.
(a) \(18.8\: \Omega \text{,}\) (b) \(18.9\: \Omega\text{,}\) (c) \(375\:\text{mA}\text{,}\) (d) \(-89.4^{\circ}\text{.}\)
Solution.
Using formulas for the corresponding quantities we get
  1. \begin{equation*} X_L = 2\pi f L = 2\pi\times 60\:\text{Hz}\times 50\:\text{mH} = 18.8\: \Omega. \end{equation*}
  2. \begin{equation*} |Z| = \sqrt{R^2 + X_L^2} = \sqrt{0.2^2 + 18.8^2} = 18.9\: \Omega. \end{equation*}
  3. \begin{equation*} I_0 = \dfrac{V_0}{|Z|} = \dfrac{\sqrt{2}V_{\text{rms}}}{Z} = \dfrac{\sqrt{2}\times 5\:\text{V}}{18.9\: \Omega} = 375\:\text{mA}. \end{equation*}
  4. \begin{equation*} \phi_I = -\tan^{-1}\left(\dfrac{\omega L}{R} \right) = -\tan^{-1}\left(\dfrac{18.8\: \Omega}{0.2\: \Omega} \right) = -89.4^{\circ}. \end{equation*}
