Exercise 2. Complex Circuit Analysis of a Pure Capacitor Circuit.

In the circuit shown in FigureÂ 41.50, find the following. (a) the current in the circuit, (b) the phase angle between the current in the circuit and the EMF, and (c) average power. Assume \(R = 0\) in the circuit.
Solution 1. a
Here, the complex impedance is \(Z = \dfrac{1}{i \omega C}\text{.}\) We write the source voltage as \(V= V_0\: e^{i\omega t}\text{,}\) where \(V_0 = 170\, \text{V}\) and \(\omega = 120\pi\, \text{sec}^{-1}\text{.}\) The current is obtained from the real part of \(I = V/Z\text{.}\)
\begin{equation*} (1)\ \ I = Re\left[ V_0\: e^{i\omega t} i \omega C \right] = V_0 \omega C\:\cos\left(\omega t + \dfrac{\pi}{2}\right) = I_0 \:\cos\left(\omega t + \dfrac{\pi}{2}\right). \end{equation*}
Putting in the numbers we find \(I_0 = V_0\omega L = 128\) mA.
Solution 2. b
The phase angle is \(\pi/2\) with the EMF lagging the current as in (1).
Solution 3. c
Since there are no dissipative elements in the circuit average power will be zero.
