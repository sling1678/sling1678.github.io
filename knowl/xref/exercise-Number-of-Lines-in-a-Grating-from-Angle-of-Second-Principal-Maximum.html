Exercise 3. Number of Lines in a Transmission Diffraction Grating from Angle of Second Principal Maximum.

How many lines per mm are there in the diffraction grating if the second order principal maximum for a light of wavelength \(536\ \text{nm}\) occurs at an angle of \(24^\circ\) with respect to the line from the grating to the center of the diffraction pattern?
Hint.
Write distance \(a\) in terms of number of lines per mm.
Answer.
\(379.4\, \text{lines per mm}\text{.}\)
Solution.
Let there be \(N\) lines per mm. Then we will have
\begin{equation*} a = \dfrac{1000}{N}\:\mu\text{m}.\ \ \ \ \ (1) \end{equation*}
From the given data of the second order peak we get
\begin{equation*} a\:\sin\:\theta_2 = 2\:\lambda.\ \ \ \ \ (2) \end{equation*}
From (1) and (2) we get
\begin{equation*} N = \dfrac{500\:\sin\:\theta_2}{\lambda} = 379.4. \end{equation*}
