Exercise 3. Complex Circuit Analysis of an RC Circuit.

A \(200\text{-}\Omega\) resistor and \(150\text{-}\mu\)F capacitor are connected in series across a \(120\text{-V}\) (rms ) AC power source oscillating at \(60\text{-Hz}\) frequency.
  1. Find the amplitude and phase of current in the circuit, assuming phase of source voltage to be zero at \(t = 0\text{.}\)
  2. Find the voltage drops across the resistor and capacitor.
  3. Find the impedance of the circuit.
  4. Find the power dissipated in the resistor.
  5. Find the power dissipated in the capacitor.
  6. Find the power produced by the source.
Hint.
Note the peak voltage \(V_0 = \sqrt{2} V_{\textrm{rms}} = 169\ \textrm{V}\text{.}\) Use complex method with \(\tilde Z = \tilde{Z}_R + \tilde{Z}_C\text{.}\)
Solution 1. a
The source voltage is given to be \(V(t) = V_0\:\cos(\omega t)\text{,}\) with \(V_0 = 120\sqrt{2} = 169.7\:\textrm{V}\) and \(\omega = 2\pi\times 60 = 120\pi\:\textrm{sec}^{-1}\text{.}\) In complex notation this is replaced by \(V_0\:e^{i\omega t}\text{.}\) Now, we divide this by complex impedance to find the complex current. The complex impedance here is
\begin{equation*} Z = R -\dfrac{i}{\omega C} = |Z|\: e^{i\phi}, \end{equation*}
where
\begin{align*} \amp |Z| = \left( R^2 + \dfrac{1}{\omega^2 C^2}\right)^{1/2} = 200.8\:\Omega,\\ \amp \phi = -\tan^{-1}\left( \dfrac{1}{\omega R C}\right)= -0.088\:\textrm{rad}. \end{align*}
Therefore, the complex current is
\begin{equation*} I = \dfrac{V}{Z} = \dfrac{V_0}{|Z|}\: e^{i(\omega t -\phi)} = (0.845\:\textrm{A})\:e^{i(\omega t +0.088)}. \end{equation*}
The real part of this complex current gives the current in the circuit.
\begin{equation*} I = (0.845\:\textrm{A})\:\cos(\omega t +0.088). \end{equation*}
Therefore, the amplitude of current is \(0.845\:\textrm{A}\) and the phase constant 0.088 rad or \(5.1^{\circ}\) ahead of the source voltage.
Solution 2. b
Now, we can use the current to find the voltage drop across the resistor.
\begin{equation*} V_R = (0.845\:\textrm{A}\times 200\:\Omega)\:\cos(\omega t + 0.088) = (169\:\textrm{V})\:\cos(\omega t + 0.088). \end{equation*}
Solution 3. c
\(Z\) already found above.
Solution 4. d
\begin{equation*} P_R = \dfrac{1}{2}I_0^2 R = \dfrac{1}{2}\times (0.845\:\textrm{A})^2\times 200\:\Omega = 71.4\:\textrm{W} \end{equation*}
Solution 5. e
No power is dissipated in the capacitor since current through the capacitor and the voltage across the capacitor are \(\pi/2\) rad out phase.
Solution 6. f
This will equal the power dissipated in the resistor: \(71.4\, \text{W}\text{.}\)
