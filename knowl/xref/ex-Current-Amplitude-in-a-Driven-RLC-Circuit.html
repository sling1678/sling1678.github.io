Example 40.28. Current Amplitude in a Driven RLC Circuit.

For each circuit in ExampleÂ 40.26, find the amplitude of the current if the peak voltage of the source of frequency \(600\text{ Hz}\) is \(10\text{ V}\text{.}\)
Answer.
(a) \(20\text{ mA}\text{,}\) (b) \(730\text{ mA}\text{,}\) (c) \(66\text{ mA}\text{,}\) and (d) \(31\text{ mA}\text{.}\)
Solution.
We use the result for a series \(RLC\) circuit.
\begin{equation*} I_0 = \dfrac{V_0}{Z},\ \ \ \ Z = \sqrt{R^2 + (\omega L - 1/\omega C)^2}. \end{equation*}
where \(\omega = 2 \pi f\) with \(f\) the frequency of the source EMF. Putting the numerical values from each figure we find the following values for the peak current in the four cases: (a) \(20\text{ mA}\text{,}\) (b) \(730\text{ mA}\text{,}\) (c) \(66\text{ mA}\text{,}\) and (d) \(31\text{ mA}\text{.}\)
