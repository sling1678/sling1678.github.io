Exercise 6. (Calculus) Solving Pressure Varying with Height Equation - Exponetially Dropping.

Solve \(\dfrac{dp}{dy} = - 3 \exp{(-2y)}\text{,}\) given \(p(0) = 1\text{ atm}\text{.}\)
Hint.
Integrate the differential equation.
Answer.
\(p(y) = \dfrac{3}{2} \exp{(-2y)} -\dfrac{1}{2}.\)
Solution.
We write the given equation as
\begin{equation*} dp = - 3 \exp{(-2y)} dy, \end{equation*}
which can be integrated to obtain
\begin{equation*} p = \dfrac{3}{2} \exp{(-2y)} + C. \end{equation*}
Now, we fit the condition at $y=0$.
\begin{equation*} 1 = \dfrac{3}{2} +C\ \ \Longrightarrow\ \ C = -\dfrac{1}{2}. \end{equation*}
Therefore
\begin{equation*} p (y) = \dfrac{3}{2} \exp{(-2y)} -\dfrac{1}{2}. \end{equation*}
