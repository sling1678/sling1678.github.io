Example 5.32. Cartesian to Polar Coordinates.

Convert the following Cartesian coordinates \((x,y)\) into polar coordinates.
  1. \((0,1)\text{;}\)
  2. \((1,0)\text{;}\)
  3. \((-1,0)\text{;}\)
  4. \((0,-1)\text{;}\)
  5. \((3,4)\text{;}\)
  6. \((-3,4)\text{;}\)
  7. \((-3,-4)\text{;}\)
  8. \((3,-4)\text{.}\)
Answer.
See solution.
Solution.
Answers are
  1. \(r = 1\text{,}\) \(\theta = \pi/2\)
  2. \(r = 1\text{,}\) \(\theta = 0\)
  3. \(r = 1\text{,}\) \(\theta = -\pi\)
  4. \(r = 1\text{,}\) \(\theta = -\dfrac{3}{2}\pi\)
  5. \(r = 5\text{,}\) \(\theta = \tan^{-1}({4/3}) = 53^\circ\)
  6. \(r = 5\text{,}\) \(\theta = \tan^{-1}({4/-3}) = -53^\circ\) clockwise from negative \(x\) axis.
  7. \(r = 5\text{,}\) \(\theta = \tan^{-1}({-4/-3}) = 53^\circ\) counterclockwise from negative \(x\) axis.
  8. \(r = 5\text{,}\) \(\theta = \tan^{-1}({-4/3}) = -53^\circ\) clockwise from positive \(x\) axis.
