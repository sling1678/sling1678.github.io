Exercise 1. Resonance Frequency of Current n RLC-Circuits.

Find the frequency at which current in each circuit will resonate.
Solution.
(a) \(\omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{10\times 10^{-3}\:\textrm{H}\times 4\times 10^{-6}\textrm{F}}} = 5000\:\textrm{sec}^{-1}\text{.}\) This give \(f_R = \omega_R/2\pi = 796\:\textrm{Hz}\text{.}\)
(b) \(\omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{1\times 10^{-3}\:\textrm{H}\times 20\times 10^{-6}\textrm{F}}} = 7071\:\textrm{sec}^{-1}\text{.}\) This give \(f_R = \omega_R/2\pi = 1125\:\textrm{Hz}\text{.}\)
(c) \(\omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{4\times 10^{-3}\:\textrm{H}\times 1.6\times 10^{-6}\textrm{F}}} = 12500\:\textrm{sec}^{-1}\text{.}\) This give \(f_R = \omega_R/2\pi = 1989\:\textrm{Hz}\text{.}\)
(d) \(\omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{5\times 10^{-3}\:\textrm{H}\times 2\times 10^{-6}\textrm{F}}} = 10000\:\textrm{sec}^{-1}\text{.}\) This give \(f_R = \omega_R/2\pi = 1592\:\textrm{Hz}\text{.}\)
