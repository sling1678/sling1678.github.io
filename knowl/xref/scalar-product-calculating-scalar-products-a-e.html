Exercise 1. Calculating Scalar Products.

Calculate the scalar product between the following pairs of vectors:
  1. \((10\ \textrm{N})\ \hat u_x\) + \((-5\ \textrm{N})\ \hat u_y\) and \((3\ \textrm{m})\ \hat u_x\) + \((4\ \textrm{m})\ \hat u_y\text{,}\)
  2. \((40\ \textrm{N})\ \hat u_x\) + \((-10\ \textrm{N})\ \hat u_y\) and \((3\ \textrm{m})\ \hat u_x\) + \((4\ \textrm{m})\ \hat u_z\text{,}\)
  3. \((100\ \textrm{N})\ \hat u_x\) + \((-10\ \textrm{N})\ \hat u_y\) and \((30\ \textrm{m})\ \hat u_y\) + \((4\ \textrm{m})\ \hat u_z\text{,}\)
  4. \((2\ \textrm{N})\ \hat u_x\) + \((3\ \textrm{N})\ \hat u_y + (4\ N)\ \hat u_z\) and \((4\ \textrm{m/s})\ \hat u_x\) + \((3\ \textrm{m/s})\ \hat u_y\) + \((2\ \textrm{m/s})\ \hat u_z\text{,}\)
  5. \((100\ \textrm{N})\ \hat u_x\) + \((-10\ \textrm{N})\ \hat u_y\) + \((100\ N)\ \hat u_z\) and \((30\ \textrm{m/s})\ \hat u_y\) + \((4\ \textrm{m/s})\ \hat u_z\text{.}\)
Answer.
See solution.
Solution.
These exercises are straightforward applications of the following formula for the scalar product: \(\vec A\cdot \vec B = A_x B_x + A_y B_y + A_z B_z\text{.}\)
(a) \(\vec A\cdot \vec B = (10\ \textrm{N})(3\ \textrm{m}) + (-5\ \textrm{N})(4\ \textrm{m}) = -7 \ \textrm{N.m}\)
(b) \(\vec A\cdot \vec B = (40\ \textrm{N})(3\ \textrm{m}) + 0 + 0 = 120\ \textrm{N.m}\)
(c) \(\vec A\cdot \vec B = (-10\ \textrm{N})(30\ \textrm{m}) + 0 + 0 = -300\ \textrm{N.m}\)
(d) 25 N.m/s.
(e) 100 N.m/s.
