Exercise 1. Direction of Position Vector from a Direction Cosine Calculation.

A position vector makes \(120^{\circ}\) with respect to the positive \(x\) axis and \(90^{\circ}\) with respect to the positive \(y\) axis. What angle does it make with respect to the positive \(z\) axis?
Hint.
Use the identity \(\cos^2\, \theta_x + \cos^2\, \theta_y + \cos^2\, \theta_z = 1\)
Answer.
\(30^{\circ},\ 150^\circ.\)
Solution.
Using the identity \(\cos^2\, \theta_x + \cos^2\, \theta_y + \cos^2\, \theta_z = 1\text{,}\) we get
\begin{equation*} \cos\,\theta_z = \pm\sqrt{1- \dfrac{1}{4} - 0 } = \pm\dfrac{\sqrt{3}}{2} \end{equation*}
Therefore, \(\theta_z = 30^{\circ},\ 150^\circ\text{.}\) We need more information about the octant the vector points to choose between the two values.
