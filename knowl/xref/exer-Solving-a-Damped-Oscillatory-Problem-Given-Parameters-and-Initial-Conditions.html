Exercise 4. Solving a Damped Oscillatory Problem Given Parameters and Initial Conditions.

A damped oscillator has following parameters \(m\) = 0.2 kg, \(k\) = 10 N/m, \(b\) = 6 kg/s. Suppose the oscillator moves on the \(x\)-axis with initial position and velocity values \(x(0) = 1\,\text{cm}\text{,}\) and \(v_x(0) = 0\text{.}\) Find \(x(t)\text{.}\)
Answer.
\(x(t) = -(0.068\ \textrm{cm})\ e^{-28 t} + (1.07\ \textrm{cm}) e^{-1.8 t}.\)
Solution.
Since this is a damped oscillator, we first need to knwo what type of dfamping is applicable here since the form of \(x(t)\) will depend on that. To decide the type, we need the values of \(\sqrt{k/m}\) and \(b/2m\text{.}\)
\begin{align*} \amp \omega_0 = \sqrt{k/m} = \sqrt{10/0.2} = \sqrt{50} = 7.1\,\text{s}^{-1}.\\ \amp \beta = \gamma/2 = \frac{b}{2m} = \frac{6}{2\times 0.2} = 15\,\text{s}^{-1}. \end{align*}
We see that \(\omega_0 \lt \gamma/2\text{.}\) Hence, it an overdamped oscillator. Therefore, \(x(t)\) will have form of the over-damped solution.
\begin{equation*} x(t) = e^{(-\beta t)}\ [ A\ e^{(-\alpha t)} + B\ e^{(\alpha t)}], \end{equation*}
where \(\beta = 15\ \textrm{s}^{-1}\) and \(\alpha = \sqrt{15^2-7.1^2} = 13.2\,\text{s}^{-1}\) . From this we get
\begin{equation*} v_x(t) = -(\alpha + \beta)A\ e^{[-(\alpha+\beta) t]} + (\alpha - \beta)B\ e^{[(\alpha-\beta) t]} . \end{equation*}
Now, we use the initial conditions on \(x\) and \(v_x\text{.}\)
\begin{align*} \amp 1 = A + B. \\ \amp 0 = -(\alpha + \beta) A + (\alpha - \beta)B. \end{align*}
Threfore,
\begin{align*} \amp A = \frac{\alpha - \beta}{2\alpha} = - 0.068\,\text{cm}\\ \amp B = \frac{\alpha + \beta}{2\alpha} = 1.07\,\text{cm}. \end{align*}
The solution
\begin{equation*} x(t) = -(0.068\ \textrm{cm})\ e^{-28 t} + (1.07\ \textrm{cm}) e^{-1.8 t}. \end{equation*}
