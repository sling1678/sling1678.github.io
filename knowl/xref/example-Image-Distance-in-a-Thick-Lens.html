Example 44.63. Image Distance in a Thick Lens.

An object of height \(1.2\,\text{cm}\) is placed \(50\,\text{cm}\) from the vertex of a \(1\text{-cm}\) thick biconvex lens having radii \(20\,\text{cm}\) and \(40\,\text{cm}\text{.}\) Suppose the lens is made of glass of refractive index \(1.5\text{.}\) Find (a) focal length, (b) principal points \(H_1\) and \(H_2\text{,}\) and (c) the image distance, orientation, and height. (d) If image is formed behind the lens, how far from that face is the image located? If the image is formed on the same side as the object, how far is the image located from the front surface?
Solution 1. a
From \(R_1\text{,}\) \(R_2\text{,}\) and \(d\text{,}\) we can compute the focal length by using the thick-lens formula for it. Let’s denote \(n_l\) for the refractive index of lens.
\begin{equation*} \frac{1}{f} = (n_l-1)\left( \frac{1}{R_1} - \frac{1}{R_2}\right) + \frac{(n_l-1)^2\,d}{n_l\,R_1\,R_2}. \end{equation*}
We have to be careful with the signs for \(R_1\) and \(R_2\text{.}\) For the given lens, \(R_1= +20\,\text{cm}\) and \(R_2 = -40\,\text{cm}\text{.}\)
\begin{equation*} \frac{1}{f} = 0.5\left( \frac{1}{20} - \frac{1}{-40}\right) + \frac{0.5^2\times 1.0}{1.5\times 20\times (-40)} = 0.03729\,\text{cm}^{-1}. \end{equation*}
Therefore, focal length is
\begin{equation*} f = \frac{1}{0.03729} = 26.8\,\text{cm}. \end{equation*}
Solution 2. b
We can find the distance of the principal points from the vertex by the given formulas in this section.
\begin{align*} \amp h_1 = -\frac{f\,(n_l-1)\, d}{ n_l\, R_2},\quad h_2 = -\frac{f\,(n_l-1)\, d}{ n_l\, R_1} \end{align*}
Thus
\begin{align*} \amp h_1 = -\frac{26.8 \times 0.5 \times 1}{ 1.5\times (-40)} = 0.223\,\text{cm}.\\ \amp h_2 = -\frac{26.8 \times 0.5 \times 1}{ 1.5\times 20} = - 0.447\,\text{cm}. \end{align*}
\(h_1\) is distance of principal point \(H_1\) from front vertex. Positive \(h_1\) means it is to the right of vertex. Simialrly, \(h_2\) is distance of principal point \(H_2\) from back vertex. Negative \(h_2\) means it is to the left of this vertex.
Solution 3. c
We can image distance from the second principal point by using the lens formula.
\begin{equation*} \frac{1}{p} + \frac{1}{q} = \frac{1}{f}. \end{equation*}
Here \(p\) is the object distance from the first principal point. But, in the problem statement, the distance of the object is given from the vertex of the lens. therefore, we must add \(h_1\) to that to get \(p\)
\begin{equation*} p = 50 + 0.223 = 50.223\,\text{ cm}. \end{equation*}
You can see it didn’t much difference here. Let’s continue to get \(q\text{.}\)
\begin{equation*} q = \frac{pf}{p-f} = \frac{50.223\times 26.8}{50.223 - 26.8} = 57.5\,\text{cm}. \end{equation*}
The transverse magnification will be
\begin{equation*} M_T = -\frac{q}{p} = -\frac{57.5}{50.223} = - 1.14. \end{equation*}
That is, image is inverted with respect the object and is \(14\%\) magnified. That gives the image height to be
\begin{equation*} |h_i| = 1.14 \times 1.2 = 1.37\,\text{cm}. \end{equation*}
Solution 4. d
In (c) we found \(q\text{,}\) which is positive. That means the image forms behind the lens at a disance \(q\) from the second principal point. The second principal point is \(h_2 = - 0.447\,\text{cm}\) from the vertex. Since it is negative, the point \(H_2\) is to the left of the vertex. That means, the distance of image from the vertex will be
\begin{equation*} s_\text{i} = d + h_2 = 57.5 - 0.447 = 56.1\,\text{cm}. \end{equation*}
