Exercise 3. Peak Current as a Function of Frequency.

For circuit (b) in Figure 40.27, plot the peak current as a function of frequency of a \(10\text{-V}\) sinusoidal source.
Solution.
We treat the peak current \(I_0\) as a function of frequency \(f\) of the driving EMF.
\begin{equation*} I_0 (f) = \dfrac{V_0}{\sqrt{R^2 + (2\pi f L - 1/2\pi f C)^2}}. \end{equation*}
Here, we have the following numerical values.
\begin{equation*} R = 10\:\Omega,\ \ L = 1\times 10^{-3}\:\textrm{H},\ \ C = 10\times 10^{-6}\:\textrm{F},\ \ V_0 = 10\:\textrm{V}. \end{equation*}
We use Mathematica® to generate the plot.
