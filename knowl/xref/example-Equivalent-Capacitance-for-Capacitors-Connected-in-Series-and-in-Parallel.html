Example 34.85. Equivalent Capacitance for Capacitors Connected in Series and in Parallel.

Determine the equivalent capacitance of the following connected capacitors. Use $C_1$ = 20 $\mu$F, $C_2$ = 30 $\mu$F, $C_3$ = 40 $\mu$F.
Answer.
(a) $50\, \mu\text{F}$, (b) $90\, \mu\text{F}$, (c) $120/13\, \mu\text{F}$, (d) $140/9\, \mu\text{F}$.
Solution.
\begin{align*} \amp \textrm{(a) } C = C_1 + C_2 = 50\:\mu\textrm{F},\\ \amp \textrm{(b) } C = C_1 + C_2 + C_3 = 90\:\mu\textrm{F},\\ \amp \textrm{(c) } C^{-1} = C_1^{-1} + C_2^{-1} + C_3^{-1} = \dfrac{1}{20} + \dfrac{1}{30} + \dfrac{1}{40} = \dfrac{13}{120}\\ \amp\ \ \ \ \ \ \ \Longrightarrow\ \ C = \dfrac{120}{13}\:\mu\textrm{F},\\ \amp \textrm{(d) } C^{-1} = C_1^{-1} + \left( C_2 + C_3 \right)^{-1} = \dfrac{1}{20} + \dfrac{1}{30+40} = \dfrac{9}{140}\\ \amp \ \ \ \ \ \ \Longrightarrow\ \ C = \dfrac{140}{9}\:\mu\textrm{F}. \end{align*}
