Example 39.27. Variation of Current in an RL-Circuit.

A \(12\text{-V}\) battery is connected to a \(10\text{-mH}\) inductor and a \(50\text{-}\Omega\) resistor at \(t = 0\text{.}\) Find the following: (a) the inductive time constant, (b) the current at \(t = 0\text{,}\) (c) the current as \(t \rightarrow \infty\text{,}\) (d) the time for the current to reach quarter of the maximum.
Answer.
(a) \(0.2\; \text{msec}\text{,}\) (b) \(0\text{,}\) (c) \(0.24\: \text{A}\text{,}\) (d) \(58\: \mu\text{s} \text{.}\)
Solution 1. a
The time constant of an RL-circuit is
\begin{equation*} \tau = \dfrac{\mathcal{L}}{R} = \dfrac{10 \text{ mH}}{50\, \Omega} = 0.2\text{\, msec}. \end{equation*}
Solution 2. b
Since a current through an inductor cannot change abruptly since that would imply infinite back EMF. That means current through an inductor is a continuous function of time. Therefore, \(I = 0\) for \(t\lt 0\) would imply that \(I=0\) at \(t=0\text{.}\)
Solution 3. c
Current reaches a steady state as \(t \rightarrow \infty\text{,}\) and there will be no induced EMF in the circuit. Setting \(dI/dt = 0\) in Faradayâ€™s flux rule yields,
\begin{align*} I_{\text{max}} \amp = \mathcal{E}_0/R \\ \amp = 12\: \text{V}/50\: \Omega = 0.24\: \text{A}. \end{align*}
Solution 4. d
To find the time to reach one-fourth of the maximum current we set \(I(t) = \dfrac{1}{4}\,I_{\text{max}}\) and solve the resulting equation for \(t\text{.}\)
\begin{equation*} \frac{I_{\text{max}}}{4} = I_{\text{max}}\left( 1 - e^{-t/\tau}\right). \end{equation*}
Simplifying this equation gives
\begin{equation*} e^{-t/\tau} = 3/4. \end{equation*}
Taking the natural logarithm of both sides yields
\begin{equation*} t = -\tau\ln{(0.75)} = 58\: \mu\text{s}. \end{equation*}
