Example 40.26. Practicing Resonance Frequency of Current in RLC Circuits.

Find frequencies at which current in each circuit of Figure 40.27 will resonate.
Answer.
(a) \(796\:\text{Hz}\text{,}\) (b) \(1125\:\text{Hz} \) (c) \(1989\:\text{Hz}\text{,}\) (d) \(1592\:\text{Hz}\text{.}\)
Solution 1. a
\begin{equation*} \omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{10\times 10^{-3}\:\text{H}\times 4\times 10^{-6}\text{F}}} = 5000\:\text{sec}^{-1}. \end{equation*}
This give \(f_R = \omega_R/2\pi = 796\:\text{Hz}\text{.}\)
Solution 2. b
\begin{equation*} \omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{1\times 10^{-3}\:\text{H}\times 20\times 10^{-6}\text{F}}} = 7071\:\text{sec}^{-1}. \end{equation*}
This give \(f_R = \omega_R/2\pi = 1125\:\text{Hz}\text{.}\)
Solution 3. c
\begin{equation*} \omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{4\times 10^{-3}\:\text{H}\times 1.6\times 10^{-6}\text{F}}} = 12500\:\text{sec}^{-1}. \end{equation*}
This give \(f_R = \omega_R/2\pi = 1989\:\text{Hz}\text{.}\)
Solution 4. d
\begin{equation*} \omega_R = \dfrac{1}{\sqrt{LC}} = \dfrac{1}{\sqrt{5\times 10^{-3}\:\text{H}\times 2\times 10^{-6}\text{F}}} = 10000\:\text{sec}^{-1}. \end{equation*}
This give \(f_R = \omega_R/2\pi = 1592\:\text{Hz}\text{.}\)
