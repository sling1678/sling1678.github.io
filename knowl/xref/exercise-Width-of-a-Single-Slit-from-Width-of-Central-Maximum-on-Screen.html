Example 48.13. Width of a Single Slit from Width of Central Maximum on Screen.

A monochromatic light of wavelength \(632.8\, \text{nm}\) incident on a single slit has central bright spot of width \(5\, \text{cm}\) at a screen \(1.6\, \text{m}\) away. Find the width of the slit.
Answer.
\(40.5\ \mu\)m.
Solution.
From the given data we can find the direction to the \(m=1\) minimum.
\begin{equation*} \theta_1 = \tan^{-1} \left( \dfrac{2.5\:\text{cm}}{160\:\text{cm}} \right) = 0.895^{\circ}. \end{equation*}
Therefore,
\begin{equation*} b\:\sin\:\theta_1 = 1\:\lambda,\ \ \Longrightarrow\ \ b = \dfrac{\lambda}{\sin\:\theta_1}. \end{equation*}
Hence,
\begin{equation*} b = \dfrac{632.8\:\text{nm}}{\sin\:0.895^{\circ}} = 40.5\:\mu\text{m}. \end{equation*}
