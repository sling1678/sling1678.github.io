Exercise 2. Interference of two Visible Light Waves.

A light of wavelength 632.8 nm wave is passed through a lens. The beam spreads out and is incident perpendicularly on an opaque board that has two very narrow slits which are separated by 3 \(\mu\)m. The emerging waves from the two slits will interfere constructively and destructively. (a) Find the directions in which the destructive interferences of lowest three orders would take place. (b) At what angle the condition for the constructive interference, based on the small angle approximation, is violated.
Solution 1. a
The constructive and destructive conditions use the ratio \(\lambda/d\text{,}\) which is
\begin{equation*} \frac{\lambda}{d} = \frac{0.6328\ \mu\text{m}}{3\ \mu\text{m}} = 0.2109. \end{equation*}
The conditions for destructive interference for the given situation are
\begin{equation*} \text{Destructive:}\ \ \sin\theta = \frac{\lambda}{d} \left(n+\frac{1}{2}\right), \ \ (n = 0 , \pm 1, \pm 2, \dots). \end{equation*}
Perhaps it is better to write this formula so that indexes are easier to work with.
\begin{equation*} \text{Destructive:}\ \ \sin\theta = \frac{\lambda}{2d}\ m, \ \ (m = \pm 1, \pm 3, \pm 5, \dots). \end{equation*}
Let us list four destructive places.
\begin{align*} \amp \sin\theta_{\pm 1} = \pm \frac{0.2109}{2}= \pm0.1055\ \ \Longrightarrow\ \ \theta = \pm 6.1^\circ.\\ \amp \sin\theta_{\pm 3} =\pm 3\times\frac{0.2109}{2}\ \ \Longrightarrow\ \ \theta = \pm18.4^\circ. \end{align*}
Solution 2. b
The constructive conditions occur when
\begin{equation*} \text{Constructive:}\ \ \sin\theta = 0.2109\ n, \ \ (n = 0 , \pm 1, \pm 2, \dots). \end{equation*}
Let us list five constructive places.
\begin{align*} \amp \sin\theta_0 = 0 \ \ \Longrightarrow\ \ \theta =0\\ \amp \sin\theta_{\pm1} = \pm 0.2109\ \ \Longrightarrow\ \ \theta = \pm 12.2^\circ.\\ \amp \sin\theta_{\pm1} = 3\times\frac{0.2109}{2}\ \ \Longrightarrow\ \ \theta = 24.9^\circ. \end{align*}
