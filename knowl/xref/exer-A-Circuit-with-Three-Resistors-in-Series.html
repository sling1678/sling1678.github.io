Exercise 1. A Circuit with Three Resistors in Series.

Find the voltages \(V_1\text{,}\) \(V_2\) and \(V_3\) across individual resistors and the current \(I\) in the circuit for the following values of resistors and voltage \(V\text{.}\) (a) \(R_1 = 2 \Omega\text{,}\) \(R_2 = 3 \Omega\text{,}\) \(R_3 = 4 \Omega\text{,}\) \(V=12\,\text{V}\text{;}\) (b) \(R_1 = 10 \Omega\text{,}\) \(R_2 = 20 \Omega\text{,}\) \(R_3 = 30 \Omega\text{,}\) \(V=100\,\text{V}\text{.}\)
Answer.
(a) \(V_1 = 8/3 \ \text{V}\text{,}\) \(V_2 = 4\ \text{V}\text{,}\) \(V_3 = 50/3 \ \text{V}\text{;}\) (b) \(V_1 = 50/3 \ \text{V}\text{,}\) \(V_2 = 100/3\ \text{V}\text{,}\) \(V_3 = 40 \ \text{V}\text{.}\)
Solution 1. a
I will work out answers for both (a) and (b) and then put numbers later. Let \(R\) be the equivalent resistance and \(I\) the current in the wire. The equivalent resistance of the circuit is
\begin{equation*} R = R_1 + R_2 + R_3.\ \ \ \ (1) \end{equation*}
Therefore current in the circuit is
\begin{equation*} I = V/R.\ \ \ \ (2) \end{equation*}
Individual voltage drops are
\begin{align*} \amp V_1 = I R_1,\ \ \ \ (3)\\ \amp V_2 = I R_2,\ \ \ \ (4)\\ \amp V_3 = I R_3,\ \ \ \ (5) \end{align*}
all using the same current given in (2).
\begin{align*} \amp R = 9\:\Omega,\ \ I = 12\:\text{V}/9\:\Omega = \dfrac{4}{3}\:\text{A}. \ \ V_1 = \dfrac{4}{3}\:\text{A}\times 2\:\Omega = \dfrac{8}{3}\:\text{V},\\ \amp V_2 = \dfrac{4}{3}\:\text{A}\times 3\:\Omega = 4\:\text{V},\ \ V_3 = \dfrac{4}{3}\:\text{A}\times 4\:\Omega = \dfrac{16}{3}\:\text{V}. \end{align*}
Solution 2. b
\begin{align*} \amp R = 60\:\Omega,\ \ I = 100\:\text{V}/60\:\Omega = \dfrac{5}{3}\:\text{A}. \ \ V_1 = \dfrac{5}{3}\:\text{A}\times 10\:\Omega = \dfrac{50}{3}\:\text{V},\\ \amp V_2 = \dfrac{5}{3}\:\text{A}\times 20\:\Omega = \dfrac{100}{3}\:\text{V},\ \ V_3 = \dfrac{5}{3}\:\text{A}\times 30\:\Omega = 40\:\text{V}. \end{align*}
