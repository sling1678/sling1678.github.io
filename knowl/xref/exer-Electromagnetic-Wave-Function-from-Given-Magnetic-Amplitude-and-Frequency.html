Exercise 5. Electromagnetic Wave Function from Given Amplitude of Magnetic Wave and Frequency.

A plane electromagnetic wave of frequency \(500\, \text{kHz}\) moves in the positive \(z\)-axis direction such that its magnetic field is pointed either towards the positive \(x\)-axis or towards the negative \(x\)-axis. The amplitude of the magnetic field is \(10^{-9}\) T. The start of time is chosen so that at \(t = 0\text{,}\) the magnetic field has an amplitude of \(10^{-9}\) T at the origin. (a) Write the wave function that will describe the magnetic field wave. (b) Find the wave function that will describe the associated electric field wave.
Solution 1. a
The amplitude of the magnetic field at \(x=0\) at \(t=0\) can be used to find the phase constant \(\phi\text{.}\) Let us write the magnetic field wave using a cosine function.
\begin{equation*} \vec B = \hat u_x\:B_0\:\cos(kz-\omega t+\phi), \end{equation*}
where
\begin{equation*} B_0 = 10^{-9}\: \textrm{T},\ \ \omega = 2\pi\times 5\times 10^{5}\:\textrm{s}^{-1},\ \ k = \dfrac{\omega}{c} = 0.0105\:\textrm{m}^{-1}, \end{equation*}
and \(\phi\) is determined from the given data at \(x=0\text{,}\) \(t=0\text{,}\)
\begin{equation*} B_0\:\cos(\phi) = 10^{-9}\:\textrm{T}, \end{equation*}
which gives \(\phi = 0\text{.}\) Note that we could have guessed this value right from the start.
Solution 2. b
The associated electric field wave will be given by
\begin{equation*} \vec E = \hat u_y\:E_0\:\cos(kz-\omega t), \end{equation*}
with
\begin{equation*} E_0 = c\:B_0 = 0.3\:\textrm{N/C}, \end{equation*}
the same \(k\) and \(\omega\) as for the magnetic field wave.
