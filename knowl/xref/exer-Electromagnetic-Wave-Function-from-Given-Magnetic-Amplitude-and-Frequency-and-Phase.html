Exercise 6. Electromagnetic Wave Function from Magnetic Wave, Frequency, and Phase.

A plane electromagnetic wave of frequency 300 MHz moves in the positive \(z\)-axis direction such that its magnetic field is pointed either towards the positive \(x\)-axis or towards the negative \(x\)-axis. The amplitude of the magnetic field is \(5\times10^{-6}\) T. The start of time is chosen so that at \(t = 0\text{,}\) the magnetic field has an amplitude \(2\times10^{-6}\) T at the origin. (a) Write the wave function that will describe the magnetic field wave. (b) Find the wave function that will describe the associated electric field wave.
Solution 1. a
We follow the procedure given in the last exercise. The amplitude of the magnetic field at \(x=0\) at \(t=0\) can be used to find the phase constant \(\phi\text{.}\) Let us write the magnetic field wave using a cosine function.
\begin{equation*} \vec B = \hat u_x\:B_0\:\cos(kz-\omega t+\phi), \end{equation*}
where
\begin{equation*} B_0 = 5\times 10^{-6}\: \textrm{T},\ \ \omega = 2\pi\times 3\times 10^{9}\:\textrm{s}^{-1},\ \ k = \dfrac{\omega}{c} = 62.8\:\textrm{m}^{-1}, \end{equation*}
and \(\phi\) is determined from the given data at \(x=0\text{,}\) \(t=0\text{,}\)
\begin{equation*} B_0\:\cos(\phi) = 2\times 10^{-6}\:\textrm{T}, \end{equation*}
which gives
\begin{equation*} \phi = \cos^{-1}\left( \dfrac{2\times 10^{-6}\:\textrm{T}}{5\times 10^{-6}\: \textrm{T}}\right) = 1.16\:\textrm{rad}. \end{equation*}
Solution 2. b
The associated electric field wave will be given by
\begin{equation*} \vec E = \hat u_y\:E_0\:\cos(kz-\omega t+\phi), \end{equation*}
with
\begin{equation*} E_0 = c\:B_0 = 1500\:\textrm{N/C}, \end{equation*}
the same \(k\text{,}\) \(\omega\text{,}\) and \(\phi\) as for the magnetic field wave.
