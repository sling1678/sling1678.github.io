Exercise 1. A Circuit with Three Resistors in Parallel.

Find the voltages \(V_1\text{,}\) \(V_2\) and \(V_3\) across individual resistors, the currents \(I_1\text{,}\) \(I_2\) and \(I_3\) through each resistor in the circuit for the following values of resistors and voltage. (a) \(R_1 = 2 \Omega\text{,}\) \(R_2 = 3 \Omega\text{,}\) \(R_3 = 4 \Omega\text{,}\) \(V=12\text{ V}\text{;}\) (b) \(R_1 = 10 \Omega\text{,}\) \(R_2 = 20 \Omega\text{,}\) \(R_3 = 30 \Omega\text{,}\) \(V=100\text{ V}\text{.}\)
Answer.
(a) \(I_1 = 6 \ \text{A}\text{,}\) \(I_2 = 4 \ \text{A}\text{,}\) \(I_3 = 3 \ \text{A}\text{;}\) (b) \(I_1 = 10 \ \text{A}\text{,}\) \(I_2 = 5 \ \text{A}\text{,}\) \(I_3 = 10/3 \ \text{A}\text{.}\)
Solution 1. a
The resistors are in parallel and same voltage drops across each resistor.
\begin{equation*} V_1 = V_2 = V_3 = 12\;\text{V}. \end{equation*}
Hence, the current through each resistor is easily obtained using Ohmâ€™s law for each.
\begin{align*} \amp I_1 = V_1/R_1 = 12\;\text{V}/2\:\Omega = 6\:\text{A},\\ \amp I_2 = V_2/R_2 = 12\;\text{V}/3\:\Omega = 4\:\text{A},\\ \amp I_3 = V_3/R_3 = 12\;\text{V}/4\:\Omega = 3\:\text{A}. \end{align*}
Solution 2. b
For the new values we get
\begin{equation*} V_1 = V_2 = V_3 = 100\;\text{V}. \end{equation*}
Therefore,
\begin{align*} \amp I_1 = V_1/R_1 = 100\;\text{V}/10\:\Omega = 10\:\text{A},\\ \amp I_2 = V_2/R_2 = 100\;\text{V}/20\:\Omega = 5\:\text{A},\\ \amp I_3 = V_3/R_3 = 100\;\text{V}/30\:\Omega = \frac{10}{3}\:\text{A}. \end{align*}
