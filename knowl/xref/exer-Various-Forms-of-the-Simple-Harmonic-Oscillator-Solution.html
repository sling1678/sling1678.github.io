Exercise 5. Various Forms of the Simple Harmonic Oscillator Solution.

The solution of a Simple Harmonic Motion can be written in three ways:
\begin{align*} x(t) \amp = C_1\ \cos(\omega t) + C_2\ \sin(\omega t) \\ \amp = A \cos(\omega t + \phi_c)\\ \amp = B \sin(\omega t + \phi_s) \end{align*}
Find the relations among \(C_1\text{,}\) \(C_2\text{,}\) \(A\text{,}\) \(B\text{,}\) \(\phi_c\text{,}\) and \(\phi_s\text{.}\)
(Actually, there is also a complex representation that is very useful: \(C\,e^{i\,\omega\,t}\text{,}\) where \(C\) is a complex amplitude. )
Solution.
Let us first work out the relations between \(C_1\text{,}\) \(C_2\text{,}\) \(A\text{,}\) and \(\phi_c\text{.}\) We can start by expanding the \(\cos(\omega t + \phi_c)\) and compare with the expanded expression.
\begin{align*} \amp A \cos(\omega t + \phi_c) = A \cos(\phi_c) \cos(\omega t) - A \sin(\phi_c) \sin(\omega t). \end{align*}
Since only the time \(t\) is a variable, and \(\cos\omega t\) is independent of \(\sin\omega t\text{,}\) we equate the coefficients of these functions in the two representations to obtain
\begin{align*} \amp C_1 = A \cos(\phi_c).\\ \amp C_2 = - A \sin(\phi_c). \end{align*}
These relations give
\begin{align*} \amp A^2 = C_1^2 + C_2^2.\\ \amp \tan(\phi_c) =- \frac{C_2}{C_1}. \end{align*}
Since \(A\) is the amplitude of the oscillation, \(A\) is positive.
\begin{equation*} A = \sqrt{C_1^2 + C_2^2}. \end{equation*}
Similarly, we find
\begin{align*} \amp B = \sqrt{C_1^2 + C_2^2}.\\ \amp \cot(\phi_s) = \frac{C_2}{C_1}. \end{align*}
Therefore,
\begin{align*} \amp B = A = \sqrt{C_1^2 + C_2^2}.\\ \amp \cot(\phi_s) = \frac{C_2}{C_1} = -\tan(\phi_c). \end{align*}
While inverting the tangent we need to be vigilant about the quadrant in which the pair \((C_1, C_2)\) falls when \(C_1\) is along the \(x\)-axis and \(C_2\) along the \(y\)-axis of an abstract \(xy\)-plane.
