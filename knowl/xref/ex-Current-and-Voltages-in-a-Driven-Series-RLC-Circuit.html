Example 40.30. Current and Voltages in a Driven Series RLC Circuit.

Consider the circuit given in FigureÂ 40.31.
(a) Find the steady state current in the circuit.
(b) Find the resonance frequency of current.
(c) Find the voltage across the resistor in the steady state.
(d) (Calculus) Find the voltage drop across the capacitor in the steady state.
(e) (Calculus) Find the back emf in the inductor in the steady state.
Answer.
(a) \(I(t) = (7.2\text{ mA})\, \cos\left( 200\,t + 1.57\right) \text{,}\) (b) \(\omega_R = \omega_0 = 1921\text{ sec}^{-1}\text{,}\) or \(306\text{ Hz}\text{.}\) (c) \((36\text{ mV})\, \cos\left( 200\,t + 1.57\right)\text{,}\) (d) \(12\text{ V} \,\sin(200\,t + 1.57)\text{,}\) (e) \(-5.76\text{ mV} \sin(200\, t + 1.57) \text{,}\)
Solution 1. (a)
Let us compute the constants we will need.
\begin{align*} \amp \gamma = \dfrac{R}{L} = 1250\text{ sec}^{-1}. \\ \amp \omega_0 = \dfrac{1}{\sqrt{RC}} = 1921\text{ sec}^{-1}. \\ \amp Z = \sqrt{R^2 + ( \omega L - 1/(\omega C) )^2 } = 1669\ \Omega. \end{align*}
The current amplitude
\begin{equation*} I_0 = \dfrac{V_0}{Z} = \dfrac{12\text{ V}}{1669\ \Omega} = 7.2\text{ mA}. \end{equation*}
The phase constant for current
\begin{align*} \phi_I \amp = \tan^{-1}\left( \dfrac{1}{\omega R C} - \dfrac{\omega L}{R } \right)\\ \amp = 1.57\text{ rad}. \end{align*}
Therefore, steady starte current at instant \(t\) will be
\begin{equation*} I(t) = (7.2\text{ mA})\, \cos\left( 200\,t + 1.57\right). \end{equation*}
Solution 2. (b)
Resonance angular frequency of current will be
\begin{equation*} \omega_R = \omega_0 = 1921\text{ sec}^{-1}. \end{equation*}
This would give the regular frequency
\begin{equation*} f_R = \dfrac{\omega_R}{2\pi} = 306\text{ Hz}. \end{equation*}
Solution 3. (c)
Voltage across resistor is simply \(IR\text{.}\)
\begin{equation*} V _R = (36\text{ mV})\, \cos\left( 200\,t + 1.57\right). \end{equation*}
Solution 4. (d)
Voltage across capacitor is a little tricky since \(V_C = q/C\) with \(q = \int I(t) dt\text{.}\)
\begin{align*} V_C \amp = \dfrac{1}{C}\int I(t) dt\\ \amp = \dfrac{1}{C}\int I_0\,\cos(\omega t + \phi_I) dt\\ \amp = \dfrac{I_0}{\omega C} \,\sin(\omega t + \phi_I) \\ \amp = 12.006\text{ V} \,\sin(200\,t + 1.57) \end{align*}
Solution 5. (e)
Back EMF by the inductor is also a little tricky since \(\mathcal{E}_\text{back} = L dI/dt\text{.}\)
\begin{align*} \mathcal{E}_\text{back} \amp = L \dfrac{dI}{dt}\\ \amp = L I_0\, \dfrac{d \cos(\omega t + \phi_I)}{dt}\\ \amp = -\omega L I_0 \sin(\omega t + \phi_I) \\ \amp = -5.76\text{ mV} \sin(200\, t + 1.57) \end{align*}
