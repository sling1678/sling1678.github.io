Exercise 4. Electromagnetic Wave Function from Given Amplitude of Electric Wave and Frequency.

A plane electromagnetic wave of frequency \(20\, \text{GHz}\) moves in the positive \(y\)-axis direction such that its electric field is pointed either towards the positive \(z\)-axis or towards the negative \(z\)-axis. The amplitude of the electric field is \(10\, \text{N/C}\text{.}\) The start of time is chosen so that at \(t = 0\text{,}\) the electric field has a value zero at the origin. (a) Write the wave function that will describe the electric field wave. (b) Find the wave function that will describe the associated magnetic field wave.
Answer.
(a) \(\hat u_z (10\ \textrm{N/C})\ \sin(419 y - 40\pi\times 10^{9} t)\text{,}\) (b) \(\hat u_z (3.33\times 10^{-8}\ \textrm{T})\ \sin(419 y - 40\pi\times 10^{9} t)\) (\(y\) in m and \(t\) in s).
Solution 1. a
The given choice of time indicates that we should write the oscillating part as sine since the sine of zero will be zero at the origin at \(t=0\text{.}\) The wave function will be
\begin{equation*} \vec E = \hat u_z\: E_0\:\sin\left(ky - \omega t\right), \end{equation*}
with
\begin{equation*} E_0 = 10\:\textrm{N/C},\ \ \omega = 2\pi\times 20\times 10^{9}\:\textrm{s}^{-1},\ \ k = \dfrac{\omega}{c} = 419\:\textrm{m}^{-1}. \end{equation*}
Solution 2. b
The magnetic field will be pointed towards the \(x\)-axis since \(\hat u_z\times \hat u_x = \hat u_y\text{,}\) which is the direction in which the wave travels.
\begin{equation*} \vec B = \hat u_x\: B_0\:\sin\left(ky - \omega t\right), \end{equation*}
with
\begin{equation*} B_0 = \dfrac{E_0}{c} = \dfrac{10\:\textrm{N/C}}{3\times 10^{8}\:\textrm{m/s}} = \dfrac{1}{3}\times 10^{-7}\:\textrm{T}, \end{equation*}
and same \(\omega\) and \(k\) as for the electric field wave.
