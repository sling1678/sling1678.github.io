Writing this as an equality we get the capacitor equation or fundamental capacitor formula.
\begin{equation} Q = C \Delta \phi,\tag{32.6} \end{equation}
which is often written as
\begin{equation*} Q = C V, \end{equation*}
using symbol \(V\) for \(\Delta \phi\text{,}\) the potential difference between the plates.
