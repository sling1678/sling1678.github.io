Kinematics deals with the basic physical quantities necessary for a description of motion. We use a properly marked three-dimensional Cartesian coordinate system and a clock to define position, displacement, velocity and acceleration of an object in precise mathematical language. The coordinate system is called the reference frame or simply the frame. The choice of origin and the directions of axes of the coordinate system are arbitrary. We will see that true vectors such as displacement and velocity are independent of this choice. The choices are made to simplify the mathematical description of the motion.
