We show below that entropy change for an isolated system is always positive semidefinite for all real processes. If a system is not isolated, it will be interacting with the environment, but by combining the system with its environment, we once again get an islated system for which entropy will, again, always be positive. So, if entropy of an interacting system goes down, the entropy of the environment must go up at least that amount so that the net change will always be a positive semidefinite.
\begin{equation*} \Delta S_\text{system} + \Delta S_\text{environment} \ge 0. \end{equation*}
Here the equality holds only for strictly reversible processes. For all other processes, the net change in entropy of an isolated system is always positive.
