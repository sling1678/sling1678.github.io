We will denote this vector by \(\vec k\text{.}\)
\begin{equation} \vec{k} = k\, \hat u,\tag{14.48} \end{equation}
where \(k\text{,}\) you may recall is related to the inverse of wavelength \(\lambda\text{.}\)
\begin{equation} k = \dfrac{2\pi}{\lambda}. \tag{14.49} \end{equation}
Vector \(\vec k\) is called wave vector. Let us denote position of an arbitrary point P\((x,y,z)\) by vector \(\vec r\) from origin to that point.
\begin{equation} \vec{r} = x\, \hat i + y\, \hat j + z\, \hat k,\tag{14.50} \end{equation}
where \(\hat i,\ \hat j,\ \hat k\) are the usual unit vectors along the Cartesian axes. Using wave vector and the position vector, we can write the wave function of a plane wave more generally as
\begin{equation} \psi(x,y,z,t) = A\cos\left( \vec k \cdot \vec r - 2\pi f t + \phi\right).\tag{14.51} \end{equation}
